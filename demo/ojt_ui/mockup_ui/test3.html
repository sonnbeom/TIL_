서류유형 마스터 0종
여기서 정의한 서류유형은 "검사기준별 서류 목록" 탭에서 여러 검사기준에 참조로 연결됩니다. "편집"에서 적용 검사기준을 체크/해제하면 해당 검사기준의 서류 목록에 승인용·공사 착수 전 기본값으로 자동 추가/제거됩니다 — 승인용/참고용 구분·근거규정·제출시점처럼 검사기준마다 달라지는 세부값은 저장 후 각 검사기준의 "서류 목록" 탭에서 조정합니다. 행을 클릭하면 이 서류를 요구하는 검사기준 목록이 펼쳐집니다 — 항목을 클릭하면 해당 검사기준으로 바로 이동합니다. "N개 검사기준에서 사용" 중인 서류유형을 삭제하면 해당 검사기준 전체에 영향을 주므로 확인 절차가 필요합니다.
